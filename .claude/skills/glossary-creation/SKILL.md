# 用語集作成スキル

このスキルは、プロジェクト固有の用語と技術用語を体系的に定義するための詳細ガイドです。

## テンプレートの参照

用語集を作成する際は、次のテンプレートを使用してください ./template.md
```

## 用語集の構成要素

### 1. ドメイン用語

**プロジェクト固有の概念**

例:
```markdown
## タスク (Task)
**定義**: ユーザーが完了すべき作業の単位

**関連用語**: サブタスク、タスクグループ

**使用例**:
- 「タスクを追加する」
- 「タスクを完了としてマークする」
```

### 2. 技術用語

**使用している技術の説明**

例:
```markdown
## TypeScript
**定義**: JavaScriptに型システムを追加した言語

**本プロジェクトでの用途**: 全てのソースコードで使用

**バージョン**: 5.x
```

### 3. 略語・頭字語

**プロジェクト内で使用する略語**

例:
```markdown
## CLI
**正式名称**: Command Line Interface

**意味**: コマンドラインインターフェース

**本プロジェクトでの使用**: ユーザーとの主要なインターフェース
```
